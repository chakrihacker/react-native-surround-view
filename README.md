# react-native-surround-view

A surround view based on svg

## Installation

```sh
npm install react-native-surround-view
```

## Usage

```js
import SurroundView from "react-native-surround-view";

// ...

<SurroundView>
  <Text>{'Press and hold'}</Text>
</SurroundView>
```
